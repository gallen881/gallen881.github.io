Windows 10 卡在歡迎無法登入

發表於2024-09-15|更新於2024-10-02
|字數總計:858|閱讀時長:2分鐘|閱讀量:

前言

前幾天正覺得 Windows 10 電腦卡頓，想到一段時間沒有關機了，於是心血來潮將筆電重新啟動。但是電腦卻一直無重新啟動，我只好直接強制關機再開機。沒想到開機登入後一直卡在歡迎的畫面轉圈圈，而這一轉就是一小時。以下是我嘗試的一些方法。

方法

  1. 重新啟動
    完全沒用，開機後還是卡在歡迎畫面。

  2. 直接放著讓他跑
    應該沒用，嘗試放了一個小時還在歡迎轉圈圈，受不了就關掉了。不知道放更久會不會就進去了。

  3. 用安全模式登入
    完全沒用，安全模式登入只要等五分鐘即可進入系統，與一小時相比迅速非常多，不過還是很慢。且使用安全模式進去後再重新用正常模式依然無法登入。

  4. 禁用硬碟
    完全沒用，我的筆電有三顆硬碟，我在 BIOS 中將兩顆硬碟禁用，只保留一顆系統碟。依然無法登入。

  5. 使用其他帳號登入
    有一點用，正好我的筆電有兩個帳號，另一個帳號十分快速就登入了。但當我要使用我主要的帳號授權管理員權限給第二個帳號時，輸入密碼後依然卡住。到這裡我覺得是軟體的問題，尤其是我主要的帳號。

  6. 回到還原點
    有一點用，回到還原點和重設是不一樣的。重設是將電腦的設定與軟體回復預設狀態，還原點則是將設定與軟體回到幾天前的狀態。沒想到回到還原點後再讓他在歡迎跑約二十分鐘就可以登入了！！這時候我會趕緊使用 Windows 的修復指令來嘗試修復電腦：

    詳細的可以參考這篇：
    https://support.microsoft.com/zh-tw/topic/使用系統檔案檢查程式工具來修復遺失或損毀的系統檔案-79aa86cb-ca52-166a-92a3-966e85d4094e

    這樣就修好了，但這只是治標不治本。我發現有時我直接重新啟動就會出問題，我必須要關機再開機。就算我都先關機再開機，過不了三四天又會在歡迎卡住。

  7. 升級 Windows 11
    我認為會卡在歡迎是軟體的問題，說不定升級系統可以把壞掉的文件洗掉，因此我開始嘗試升級 Windows 11。幸運的是，我的電腦符合規格，不用另外做其他事情就能升級。不幸的是，不知道出了甚麼問題，再升級時一樣遇到了卡死的問題。我使用了各種方法皆是如此：

    • Windows Update：卡在 96%
    • Windows 11 安裝小幫手：卡在 99%
    • Windwos 11 安裝程式：卡在 90%

    只剩製作開機隨身碟升級的這個方法沒有試，但以上三個方法每個都等了六個小時以上，再等下去會影響到我的事情，因此沒有繼續升級。

結論

目前已第六個方法效果最為顯著，但我並不認為這個問題就此解決了，這依舊只是一個暫時性的解決方案。如果有任何建議或想法，歡迎留言討論，我需要你的幫助。
