取得 GitHub Education benefits

什麼是 GitHub Education benefits

GitHub Education benefits 是 GitHub 給學生的優惠，使用後你可以取得 GitHub PRO 和其他各種好處。如免費註冊 .me 網域一年、免費使用 JetBrains、免費使用 GitHub Copilot 等等，個人認為有 GitHub Copilot 就值了。詳細可以到官方網站細看。

申請的前提

  • 擁有一個 GitHub 帳號
  • 13 歲以上
  • 正在受認可的機構就讀
  • 可提供在學證明的文件

ヾ(≧▽≦*)o
基本上國高中生和大學生都可以申請！！

申請步驟

  1. 前往官方頁面並登入後依照自己的需求選擇。這裡我選擇學生。

  2. 往下滑動，找到 Application。你可能需要輸入學校配發的電子郵件信箱，我已經填寫過所以並無出現。接著輸入你的學校名稱後按下 Continue。

  3. 待頁面跳轉後，準備好自己的證明文件。

    文件上必須標有日期！

  4. 往下滑動，上傳身分證明文件後點擊 Process my application。

    上傳文件請勿修改或編輯，如果編輯過會被發現，無法申請！

  5. 接著就可以等待申請通過囉！！
    印象中 1~3 天就可以通過。

申請通過後

可以到自己的 GitHub 頁面查看，可以發現左下角多了一個 PRO 的標誌，代表已經取得 GitHub PRO 了。同時也可以享有 GitHub Education benefits 的福利。🎉🎉
